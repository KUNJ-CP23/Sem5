# 📚 Semester 5 Lab Repository

Welcome to my Semester 5 lab repository. This repository contains lab implementations for the following subjects:

- 🧠 Design and Analysis of Algorithms (DAA) – implemented in **C**
- 📊 Data Mining – implemented in **Python** using various libraries
- 🔗 Graph Theory – implemented in **Java**
- 🌐 ASP.NET Core – developed in **C#**

---


## 💡 Subjects Overview

### 1. Design and Analysis of Algorithms (DAA) - C
- Focused on algorithm design paradigms such as:
  - Greedy Algorithms
  - Divide and Conquer
  - Dynamic Programming
  - Backtracking and Branch & Bound
- Implemented in **C** language with clear function-based structure.

### 2. Data Mining - Python
- Implemented key data mining techniques using Python libraries:
  - **NumPy**, **Pandas**, **Matplotlib**, **Scikit-learn**, **MLxtend**
- Covers:
  - Association Rule Mining (Apriori)
  - Classification (Decision Trees, k-NN)
  - Clustering (k-Means)
  - Data Preprocessing & Visualization

### 3. Graph Theory - Java
- Java programs for:
  - BFS and DFS
  - Dijkstra’s Algorithm
  - Topological Sorting
  - Minimum Spanning Tree (Prim’s and Kruskal’s)

### 4. ASP.NET Core - C#
- A basic web application using **ASP.NET Core MVC**
- Features:
  - MVC Architecture
  - Routing, Models, and Controllers
  - Razor Views
  - Basic CRUD Operations using Entity Framework Core
